# HDLBits Verilog Solutions

This repository contains my solutions to problems from HDLBits.
The goal is to strengthen fundamentals in digital design and RTL coding using Verilog.

## Topics Covered

* Combinational logic (basic gates, multiplexers, encoders, decoders)
* Sequential logic (flip-flops, counters, shift registers)
* Finite State Machines (FSM – Mealy and Moore)
* Arithmetic circuits (adders, subtractors)
* Memory elements (RAM, registers)

## Folder Structure

* `combinational/` → basic logic and mux problems
* `sequential/` → flip-flops, counters, shift registers
* `fsm/` → state machine designs
* `arithmetic/` → adders and related designs

## Coding Style

* Synthesizable RTL
* Clear naming conventions
* Modular design approach

## How to Run

1. Copy the Verilog code
2. Simulate using tools like ModelSim, QuestaSim, or Xilinx ISE/Vivado
3. Verify outputs using testbench or waveform

## Goal

To build strong fundamentals in:

* Verilog coding
* Digital design concepts
* Problem-solving for VLSI interviews
